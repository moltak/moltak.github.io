StackAI

이거 참 좋다. 내가 원하는거랑 거의 비슷

  • 두개의 화면이 있다. agent 설계 / 배포

  • Agent 설계 화면은 일반적임. 우리가 아는 일반적인 모습과 비슷

  • 데이터 처리 화면 하나만 더 있으면 좋은데. 약간 아쉽지만. 전처리는 다른거 써도 되니 일단. 좋아보임
